Apa itu PowerShell?

Pengertian PowerShell

PowerShell merupakan jenis command line interface yang dan sebuah teknik pemrograman berorientasi objek untuk sistem operasi Windows.

PowerShell banyak digunakan oleh para administrator yang setiap hari selalu berurusan dengan Windows Server dalam melakukan berbagai pengaturan jaringan untuk server yang berbasis Windows. PowerShell menyediakan fitur API atau Application Program Interface. yang bisa diakses oleh aplikasi yang lainnya.

Perlu Anda ketahui bahwa PowerShell merupakan sebuah produk keluaran dari .NET Framework yang bisa dijalankan oleh beberapa tipe sistem operasi. Produk ini kemudian akan menghasilkan keluaran atau output berformat .net dan .text.

Mungkin Dari Kalian Ada Yang Ingin Mengubah Thema Dan Agar Tidak Bosan,
Nah Maka dari Itu Saya Akan Memberi Tutorial Bagimana Cara Merubah Thema Oh My Posh,

Jika Ingin Melihat Thema Themanya Kalian Bisa Mengunjungi Webnya OhMyPosh Langsung Loh Ini Dia Website Nya Untuk Memilih Thema Kalian Sendiri Oh-My-Posh

Langsung Saja Ke Tutorialnya :

  1. Pilih Thema Kalian Lalu Download Kalian Di Oh My Posh

  2. Lalu Klik Saja Thema nya Nanti Kalian Akan Di Arahkan Ke Github

  3. Lalu Klik Raw Setelah Masuk Kalian Klik CTRL + S untuk download file .json tersebut

  4. setelah di download kalian taruh bebas di mana saja “Saya Akan Menyontohkan, Saya Akan Taruh Di Folder WindowsPowerShell” contoh: “C:\Users\ZiddPy\Documents\PowerShell”

  5. Setelah itu copy file location tersebut contoh: “C:\Users\ZiddPy\Documents\PowerShell\agnoster.omp.json”
    “agnoster.omp.json” adalah nama file thema oh my posh yang telah kita download tadi
